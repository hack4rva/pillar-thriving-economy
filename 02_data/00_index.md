# Data Index - Thriving Economy

This folder catalogs datasets and public sources that may help teams build prototypes related to procurement, small business navigation, and workforce/trades pathways.

## Use data carefully
Before using a dataset, ask:
- what is the source and update cadence?
- how current is it and what does it actually measure?
- what does it not measure or exclude?
- does it contain any sensitive information?
- is it suitable for a weekend prototype?

## Recommended process
1. pick a problem first
2. identify the minimum data needed
3. prefer simple, public, understandable sources
4. document assumptions and limitations
5. label demos clearly when using sample or partial data

## Data categories (examples)
- City procurement and contract data (e.g., City Contracts Registry)
- Vendor/solicitation listings from the public portal
- Small business guidance and process documents
- Workforce and training program directories
- State and federal labor/economic indicators (for context only)
