# Meta Prompt Templates

Purpose
- Templates for generating a full Pillar Research Prompt Package for any pillar repository. Clone and adapt to other pillars.

How to use
- Copy `pillar_research_prompt_package_meta.txt` into a new pillar repo and run via the Perplexity runner.
- Optionally adjust the sibling YAML (`pillar_research_prompt_package_meta.yaml`) to set model, preset, and instructions.

Notes
- Prompts should prioritize repository-provided materials and clearly mark unknowns or inferences.
- Avoid asking the model to include URLs in text; rely on API metadata.
